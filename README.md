# Swiper, No Swiping: Measuring High Quality Interception Opportunity Creation

We quantify how often defenders create interception opportunities by controlling space at the ball’s arrival, not just recording turnovers.

The full submission can be viewed on Kaggle [here](https://www.kaggle.com/competitions/nfl-big-data-bowl-2026-analytics/writeups/HQIOs)
